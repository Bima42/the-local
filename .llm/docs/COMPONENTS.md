# UI Components Reference

## Layout & Structure
- **card** - Content containers with header/body/footer
- **separator** - Visual dividers between content
- **sidebar** - App navigation panel
- **tabs** - Switch between content views
- **accordion** - Expandable/collapsible sections
- **collapsible** - Single expand/collapse container

## Navigation
- **navigation-menu** - Top-level nav with dropdowns
- **dropdown-menu** - Contextual action menus (right-click, kebab menus)
- **command** - Search/command palette (âŒ˜K style)

## Feedback & Overlays
- **dialog** - Modal for focused tasks, confirmations
- **alert-dialog** - Destructive/important confirmations (requires action)
- **sheet** - Slide-out panel (forms, details, mobile nav)
- **popover** - Floating content anchored to trigger
- **tooltip** - Hover hints for icons/truncated text
- **sonner** - Toast notifications
- **skeleton** - Loading placeholders

## Forms
- **form** - Form wrapper with validation (react-hook-form)
- **input** - Text input
- **textarea** - Multi-line text
- **checkbox** - Boolean/multi-select
- **switch** - Toggle on/off settings
- **label** - Form field labels

## Actions
- **button** - Primary actions, submissions

## Data Display
- **table** - Tabular data
- **badge** - Status indicators, tags, counts
- **avatar** - User/entity images with fallback

## Utilities
- **theme-toggle** - Dark/light mode switch
- **language-switcher** - i18n locale selector
